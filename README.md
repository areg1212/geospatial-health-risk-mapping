# Geospatial Environmental Health Risk Mapping

## Project Overview
This project explores how environmental and population factors can influence spatial health vulnerability.

The objective is to create a geospatial risk index combining:

- population density
- climate variables
- land use patterns

## Data Sources

- WorldPop (population density)
- WorldClim (climate data)
- Copernicus Land Monitoring

## Tools

- Python
- GeoPandas
- Pandas
- Folium

## Outputs

- Geospatial risk maps
- Spatial vulnerability analysis
